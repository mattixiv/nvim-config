# My Neovim configuration

A LazyVim-based configuration, mostly for React development with TypeScript

## Features

- **LSP:** `vtsls` for TypeScript and React
- **Tests:** with `neotest-jest`
- **Formatting:** Prettier
- **Git:** Gitsigns (inline blame) and LazyGit

## Usage

1. Clone the repo `git clone <url> ~/.config/nvim`
2. Install dependencies: `brew install lazygit ripgrep fd`
3. Open nvim and wait for Lazy to install plugins
