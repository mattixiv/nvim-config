# My Neovim configuration

A LazyVim-based configuration for React/TypeScript and C++/OpenGL development.

## Features

- **LSP:** `vtsls` for TypeScript/React, `clangd` for C++, `glsl-analyzer` for GLSL shaders
- **Snippets:** via `friendly-snippets`
- **Tests:** with `neotest-jest`
- **Formatting:** Prettier, clang-format
- **Git:** Gitsigns (inline blame) and LazyGit
- **AI:** GitHub Copilot and Claude Code

## Usage

1. Clone the repo: `git clone <url> ~/.config/nvim`
2. Install dependencies: `lazygit ripgrep fd clang`
3. Open nvim and wait for Lazy to install plugins
